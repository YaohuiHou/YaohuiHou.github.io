# MarkDown 编辑高亮

主要是为了在引入一些代码的时候能够高亮显示，这样界面可以更加友好

没有使用：

```
# 创建一个 markdown 文件
echo '# Hello VuePress' > README.md

```

使用之后

```bash
# 创建一个 markdown 文件
echo '# Hello VuePress' > README.md
```

语法

````
```key
代码段
```
````
